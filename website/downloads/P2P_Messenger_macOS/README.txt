P2P Messenger - Інструкція з використання
=========================================

ВСТАНОВЛЕННЯ:

1. Розпакуйте завантажений ZIP-архів у зручне місце (наприклад, на робочий стіл)
2. Встановіть Docker Desktop з сайту: https://www.docker.com/products/docker-desktop/
3. Запустіть Docker Desktop і дочекайтесь повного старту (іконка Docker у треї має бути активною)

ЗАПУСК (є два способи):

СПОСІБ 1 - Швидкий запуск:
1. Двічі клацніть на файлі 'start.command' у розпакованій папці
2. Якщо macOS блокує запуск, перейдіть в System Preferences > Security & Privacy 
   та натисніть "Open Anyway"
3. Дочекайтесь запуску додатку у браузері

СПОСІБ 2 - Створення постійного додатку (рекомендовано):
1. Двічі клацніть на файлі 'Create Launcher.command' у розпакованій папці
2. Це створить файл 'P2P Messenger.app' в тій же папці
3. Перетягніть створений файл 'P2P Messenger.app' в Applications або на Dock
4. Тепер ви можете запускати P2P Messenger як звичайний macOS додаток

ВАЖЛИВО: При першому запуску Docker буде завантажувати необхідні компоненти,
         що може зайняти до кількох хвилин.

УСУНЕННЯ ПРОБЛЕМ:
- Перезапустіть Docker Desktop, якщо виникають помилки
- Перший запуск займає більше часу через завантаження образів
- Переконайтесь, що порт 8000 не зайнятий іншими програмами
- Якщо браузер не відкрився автоматично, відкрийте http://localhost:8000

ЗУПИНКА:
1. Введіть 'exit' у консолі програми
2. Або закрийте термінал і виконайте в новому терміналі команду:
   cd "шлях до розпакованої папки" && docker-compose stop

ЗБЕРЕЖЕННЯ ДАНИХ:
Усі ваші повідомлення, контакти та налаштування автоматично зберігаються
в Docker і будуть доступні при наступному запуску програми.
